# ScamGuard Platform v1.1.0 — User Flows / Stories

## Product intent
This release shifts the product from static pages to complete user stories. It demonstrates how ScamGuard works in the background: unknown call → AI screening → decision → family notification → report update.

## Added
- New **Flöden / Stories** module.
- Four complete mock scenarios:
  - Scam call blocked.
  - Safe call passed through.
  - Family alert and review.
  - Senior mode reassurance.
- `/api/stories` mock API endpoint.
- Timeline, evidence tags, risk score, phone-style mock panel and demo playback.

## Product notes
This follows the Product Bible principle that ScamGuard is not an app the user actively uses every day. It is a protection system, and the app is the control room and proof layer.
