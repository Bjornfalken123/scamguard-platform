# v1.0.0 — Control Center

## Added
- Control Center som användarens huvudsakliga kontrollrum.
- Senior-läge och anhörig-läge som produktprincip.
- Mockad telefonvidarekoppling, skyddsnivåer, AI-inställningar och integritet.
- Anhörigkedja med roller, prioritet och varningsnivåer.
- `/api/control-center`.

## Product notes
Control Center ersätter traditionella inställningar. Det ska kännas som att användaren styr ett trygghetssystem, inte konfigurerar en teknisk app.
