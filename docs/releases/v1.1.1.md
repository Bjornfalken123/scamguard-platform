# ScamGuard Platform v1.1.1 — Simplicity Correction

## Product correction
This release corrects the product direction after v1.1.0 placed too much emphasis on relatives as active participants.

ScamGuard should feel like an automated protection service where the engine does nearly all the work. The app is a calm control panel for status, reports and simple settings.

## Added / changed
- Simplified navigation.
- Reframed relatives as report receivers and backup, not daily operators.
- Reframed stories as background flows rather than tasks for relatives.
- Added clearer language around the engine doing the work.
- Focused the app around Overview, Reports, Settings and Calls.

## Product principle
ScamGuard should reduce responsibility for both seniors and relatives. The core promise is: the user should not need to think about phone scams.
