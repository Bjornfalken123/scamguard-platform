# ADR 0001 — PWA first

## Beslut
ScamGuard byggs först som PWA/webbaserad kontrollpanel.

## Varför
Det ger snabb iteration, enkel delning, enklare användartester och mindre beroende av App Store/TestFlight i prototypfasen.

## Konsekvens
Native iOS/Android byggs senare när produktflödena är validerade.
